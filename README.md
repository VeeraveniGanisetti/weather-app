# Weather App
A weather web application built using Django and Python.
## Features
- Search weather by location
- Fetch weather information using API
- Simple and responsive interface
## Technologies Used
- Python
- Django
- HTML
- CSS
- Requests
## Installation
Clone repository:
```bash
git clone https://github.com/VeeraveniGanisetti/weather-app.git
```
Install dependencies:
```bash
pip install requests
pip install django
```
Run migrations:
```bash
python manage.py migrate
```
Start server:
```bash
python manage.py runserver
```
Open browser:
```text
http://127.0.0.1:8000/
```
